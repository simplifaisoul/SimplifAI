# SimplifAI-1 Portfolio Website

A modern, responsive portfolio website for SimplifAI-1 showcasing AI solutions and services.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: 
  - Smooth scrolling navigation
  - Mobile-friendly hamburger menu
  - Contact form with validation
  - Scroll-triggered animations
  - Typing effect for hero title
- **SEO Optimized**: Semantic HTML5 structure
- **Fast Loading**: Optimized CSS and JavaScript

## Structure

```
SimplifAI/
├── index.html          # Main HTML file
├── styles.css          # CSS styling
├── script.js           # JavaScript functionality
├── images/             # Image assets
├── assets/             # Additional assets
└── README.md           # This file
```

## Sections

1. **Hero**: Eye-catching introduction with call-to-action
2. **Services**: Overview of AI development services
3. **Portfolio**: Showcase of completed projects
4. **About**: Company information and mission
5. **Contact**: Contact form for inquiries

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser
3. The website is ready to use!

## Customization

- Edit `styles.css` to modify colors, fonts, and layout
- Update content in `index.html`
- Add custom functionality in `script.js`
- Place images in the `images/` folder
